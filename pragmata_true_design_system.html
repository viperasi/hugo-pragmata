05   CSS DESIGN TOKENS · 可复用属性
bg-blueprint
#e8eae6 + grid repeating linear-gradient 28px
工程图纸底色 — 不是黑色！浅灰+网格是整套风格的基础
corner-reg
background: linear-gradient lines 四角各 8px 十字
正交注册标记 — 模仿设计图四角对齐符号，用 background 伪造无需额外元素
yellow-accent
#e8a020 — border-left 4px / header-bg / btn-primary
工程黄 = 靴子卡扣色 = 结构性强调，不是装饰色。CTA 首选
control-marks
圆形 badge: border-radius:50% + red #c03020 / blue #2860c0
来自胸甲圆形按钮 — 用于状态标记、标签、操作锚点
sleeve-stripe
red / white / blue 三色等分水平条纹，高 3-5px
袖口条纹 → 分隔线、section divider。比单线更有识别度
ann-line
stroke-dasharray: 4,3 + rgba(60,80,100,.3) + arrowhead
标注引线 — 出现在 tooltip、图表说明、设计图解说中
sidebar-active
border-left: 3px solid #e8a020 + bg: var(--bg)
黄色左边条激活 — 对应靴子/卡扣的工程黄结构定位线
SYNTHESIS   Pragmata 的真实视觉语言 = NASA工程蓝图 × Apollo 时代宇航服美学 × 苏联工业控制面板。 浅灰底色·工程网格·黄色结构锁扣·红蓝圆形控制钮·袖口三色条纹·四角注册标记 —— 这六个元素组合在一起，才是 Pragmata 的设计语言。
