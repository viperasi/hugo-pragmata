# Hugo Pragmata

A Hugo theme inspired by NASA blueprint drawings, Apollo spacesuit panels, and Soviet control room aesthetics. Clean engineering-grade typography on a grid-paper background.

**Demo:** [https://www.xu81.com](https://www.xu81.com)

![Hugo](https://img.shields.io/badge/Hugo-0.112%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## Features

- **Blueprint grid background** — engineering drawing aesthetic throughout
- **Three-font system** — Space Mono (code/labels), Barlow Condensed (display/headings), Inter (body)
- **Responsive layout** — sidebar on desktop, hamburger menu on mobile
- **Tag system** — tag cloud with weighted sizing and color coding
- **Projects section** — dedicated project showcase grid
- **Table of contents** — auto-generated sidebar TOC for long articles
- **Breadcrumb navigation** — contextual path on every page
- **Git info** — last commit date and hash per page (with `.GitInfo`)
- **Social links** — GitHub, Gitee, Weibo, Bilibili, RSS
- **Custom 404 page** — themed error panel
- **SEO-ready** — Open Graph meta, canonical URLs, RSS feed

## Installation

```bash
cd your-hugo-site
git clone https://github.com/viperasi/hugo-pragmata.git themes/hugo-pragmata
```

Then set the theme in your site config:

```bash
echo "theme = 'hugo-pragmata'" >> hugo.toml
```

## Configuration

```toml
baseURL = "https://example.com/"
title = "YOUR SITE TITLE"
theme = "hugo-pragmata"
languageCode = "en"

enableGitInfo = true

[params]
  [params.git_info]
    host = "github"     # github or gitee
    user = "your-username"
    repo = "your-repo"

  [[params.social]]
    name = "GitHub"
    url  = "https://github.com/your-username"

  [[params.social]]
    name = "Bilibili"
    url  = "https://space.bilibili.com/your-id"

[[menu.main]]
  name = "Posts"
  url  = "/posts/"

[[menu.main]]
  name = "Projects"
  url  = "/projects/"

[[menu.main]]
  name = "Tags"
  url  = "/tags/"
```

```
│   ├── tags/
│   │   ├── list.html          # tag cloud
│   │   └── term.html          # posts by tag

```

## Design Tokens

The theme uses CSS custom properties for all visual constants. Override them to customize:

```css
:root {
  --bg:       #e8eae6;   /* main background */
  --bg2:      #dfe1dd;   /* nav/sidebar background */
  --text:     #1a2230;   /* primary text */
  --yellow:   #e8a020;   /* accent yellow */
  --red-btn:  #c03020;   /* alert red */
  --blue-btn: #2860c0;   /* link blue */
  --stripe-r: #c03020;   /* divider stripe red */
  --stripe-b: #2060b0;   /* divider stripe blue */
}
```
